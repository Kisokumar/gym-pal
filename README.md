Make sure you have `node`, `git` and `npm` installed before beginning.

```bash
# clone repo
git clone https://github.com/Kisokumar/gym-pal
cd gym-pal

# install server packages
cd server
yarn install --silent

cd ..

# install client packages
cd client
yarn install --silent
```
